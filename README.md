# Real-Time Online Lecture Attentiveness Monitoring System

An AI-powered attentiveness monitoring system designed for online lectures and virtual learning environments using Computer Vision, Machine Learning, and Facial Landmark Detection.

---

## Project Overview

This project helps monitor student attentiveness during online lectures through real-time video analysis. The system detects focus levels, analyzes facial behavior, and generates live attentiveness insights through an interactive dashboard.

The application combines a Flask backend with a React frontend to create a smart AI-based monitoring platform for virtual learning and remote education.

---

## Features

- Real-time attentiveness detection
- Online lecture monitoring
- AI-based facial landmark tracking
- Dashboard analytics and attention graphs
- Attention percentage monitoring
- Student engagement visualization
- Modern React-based UI
- Backend API integration using Flask
- Machine Learning model integration
- Real-time video stream processing
